# Movies-for-iOS
Search and discover for popular, upcoming and top rated movies

## Installation ##
* Git and clone <br/>
```bash
$ cd /Users/user/my_project
$ git clone https://github.com/TarokhDev2020/Ratios-for-iOS.git
$ pod install
```
